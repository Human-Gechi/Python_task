# Data Engineering Python Tasks

This repository contains three Python projects demonstrating automation, data analysis, and     Jira automation for DE launchpad program.

---

## Project 1: ZenQuotes API – Automated Quote Emails
Automates sending motivational quotes to subscribed users. Fetches daily quotes from [ZenQuotes API](https://zenquotes.io/) and sends personalized emails. Supports daily and weekly schedules, with logging of delivery and errors.

**Folder:** [Zenquotes](./Zenquotes)

---

## Project 2: Airbnb Data Deep Dive – Python + Pandas
Analyzes Paris Airbnb listings to uncover pricing patterns, availability trends, and host concentration. Cleans and enriches data, performs aggregation, and generates key insights.

**Folder:** [Airbnb_Analysis](./Airbnb_deep_dive)

---

## Project 3: Jira Automation – Phone Request Processing
Automates processing of phone requests from PostgreSQL into Jira Service Desk tickets. Should Handle database connections, ticket creation, and logging of errors and successes.


**Folder:** [Cowjack](./Cowjack)
